Yolo-v5 model trained on the Mars Rock dataset (https://universe.roboflow.com/benoit-putzeys/rocks-uyoff)
Includes best.pt, and the files obtained after converting it to a .blob using http://tools.luxonis.com/

Input size is 416x416. Trained for 100 epochs. Test set results (52 images, 165 objects):
P: .955, R: .778, mAP@.5: .881, mAP@.5-.95: .514
